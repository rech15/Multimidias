Tag Picture>

Esse elemento é usado para especificar múltiplos elementos source para um elemento específico img contido nele, assim o navegador vai escolher a imagem mais adequada de acordo com o layout atual da página, a habilidade do navegador de renderizar um certo tipo de imagem, se caso não houver correspondência entre os elementos source, o arquivo especificado pelo elemento img será selecionado e assim a imagem selecionada é então exibida no espaço ocupado pelo elemento img.
